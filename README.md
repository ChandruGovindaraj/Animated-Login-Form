# Modern Login & Registration Form UI

This is a modern, responsive **Login and Registration Form User Interface (UI)** built with a clean dark theme. The project is designed using **HTML, CSS, and Vanilla JavaScript**, featuring a smooth animated slider transition between the **Login** and **Sign Up** panels.  

Its professional layout and user-friendly interface make it an excellent template for authentication sections in websites and web applications.
